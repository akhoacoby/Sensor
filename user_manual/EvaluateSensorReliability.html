Evaluate Sensor Reliability

Function: bool evaluateSensorReliability(sensorId, radius, start, end)

Description

Checks if a sensor behaves consistently compared to nearby sensors over a given time period.

Parameters

  • sensorId: ID of the sensor to evaluate.
  • radius: Radius (in km) within which to consider neighboring sensors.
  • start, end: Date range in YYYY-MM-DD hh:mm:ss.

Auxiliary Functions Used

  • computeDistance(lat1, lon1, lat2, lon2): Euclidean approximation × 111 km.
  • accumulate() and sqrt() for mean and standard deviation.

Logic

  • Compare mean value of the sensor to neighbor averages for each pollutant.
  • If the value deviates more than 2 × stdDev, it's flagged unreliable.
