Analyze Sensor

Function: bool analyzeSensor(const std::string& sensorId)

Description

Determines if a sensor is reliable based on:

  • Temporal regularity: checks for consistent measurement intervals (~1 min).
  • Value variance: ensures readings are neither too constant nor too erratic.

Parameters

  • sensorId: ID of the sensor to analyze (e.g., "S1").

Auxiliary Functions Used

  • Inline computation of variance and interval checking.

Thresholds Used

  • maxIrregularRatio = 0.2: Max allowed proportion of irregular intervals.
  • minVariance = 0.1, maxVariance = 100.0: Value variance bounds.
