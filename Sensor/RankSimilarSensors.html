Rank Similar Sensors

Function: vector<pair<Sensor, double>> rankSimilarSensors(sensorId, start, end, timestamp, attributeId)

Description

Ranks other sensors by similarity to a reference sensor's measurements using Pearson correlation.

Parameters

  • sensorId: ID of the reference sensor.
  • start: Start time in ISO format (e.g., "2024-04-01 00:00:00").
  • end: End time (same format).
  • timestamp: [Unused] Reserved parameter, can be removed or clarified.
  • attributeId: Type of measurement (e.g., "O3", "PM10").

Auxiliary Functions Used

  • pearsonCorrelation(x, y): Computes similarity between vectors.

    • The Pearson correlation coefficient is a statistical measure that evaluates the linear relationship between two datasets. It produces a value between -1 and 1: a value close to 1 indicates a strong positive correlation, -1 a strong negative correlation, and 0 no correlation. In our system, it is used to compare sensor measurements over time. A high correlation suggests that sensors are behaving similarly, helping assess their reliability.

Output

A sorted list of sensors and their similarity score to the reference (from most to least similar).
