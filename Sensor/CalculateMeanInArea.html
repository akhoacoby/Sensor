Calculate Mean in Area

Function: double calculateMeanInArea(lat, lon, radius, start, end, attributeId)

Description

Calculates the average pollutant value in a geographic region within a time range.

Parameters

  • lat, lon: Coordinates of the center point.
  • radius: Maximum distance from center to include sensors (approx. in degrees).
  • start, end: Time range to filter measurements.
  • attributeId: Target attribute (e.g., "NO2").

Auxiliary Functions Used

  • Inline distance calculation using sqrt(dx² + dy²).

Output

Returns the average of all valid values. Returns -1 if no valid measurements.
